# ydm

Small script to download and tag music from youtube

To install, just allow execution and put the script in the directory that is in `$PATH`.


#### Example:
```
sudo chmod +x ydm
mv ydm /usr/local/bin
```


#### Requirements:
+ youtube-dl
+ taffy
